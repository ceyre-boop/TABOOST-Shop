# sounds/

Place the hero intro sound here as either:

- **`netflix-tudum-sfx-n-c (1).mp3`** — the original filename (already wired as first priority)
- **`netflix-tudum.mp3`** — shorter alias (also wired as second priority)
- **`hero-tudum.mp3`** — generic fallback name

The page tries each name in order and plays the first one it can load.
If none are found it falls back to a synthesised sound.
