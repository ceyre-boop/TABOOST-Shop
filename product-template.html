📋 How to use this tool
  1. Filter and select the active campaign this product belongs to.
  2. Paste the TikTok Product ID — image auto-loads from the cache.
  3. Enter the product name — exact & near matches in live data are shown instantly.
  4. Fill in details. Commission/Price auto-format on blur. Category auto-suggests.
  5. Click Generate Row (or Ctrl/⌘+Enter) and add it to the queue.
  6. Repeat for each product. Then use Copy All Rows to copy everything at once.
  7. Paste rows into data/shop/tap-products.csv and run python3 update_discovery_data.py.
🔁 Reusing an existing listing

If the checker finds an exact match the form is pre-filled automatically — just change the campaign. Near-matches are shown too so you can pick the closest one and adjust. Only use near-matches if the product is truly identical.
