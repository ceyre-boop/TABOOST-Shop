📋 How to use this tool
  1. Select the active campaign this product belongs to.
  2. Paste the TikTok Product ID — the tool checks for an existing image in the cache.
  3. Enter the product name — the tool checks for an exact match in current live data. If one is found, you can reuse its details.
  4. Fill in remaining details. Category auto-suggests from the product name keywords.
  5. Click Generate CSV Row and copy the output.
  6. Paste the row into data/shop/tap-products.csv (open in Excel/Google Sheets and paste as a new row).
  7. After adding all new products, run python3 update_discovery_data.py to regenerate js/product-data.js.
🔁 Reusing an existing listing

If the Duplicate Checker finds an exact match (same product name), the existing data will be pre-filled. Update only the Campaign ID field (select a new campaign above) and regenerate the row. The product's original image, category, rank, and sales data will be preserved.
